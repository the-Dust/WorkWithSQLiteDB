ИНСТРУКЦИЯ

1 Распаковать архив 

2 Запустить файл Start.bat, после этого в данную папку скомпилируется файл Program.Exe

3 Запустить консоль, с помощью команды cd установить рабочий каталог. В этот каталог будет записана база данных.

	Пример: cd D:\thisFolder

4 Запустить файл Program.Exe из консоли, указав в качестве аргумента название текстового файла (согласно заданию; 
текст задания прилагается). Не забыть указать полный путь до обоих файлов.
  
	Пример: D:\thisFolder\Program.Exe D:\thisFolder\Example.tsv

	Если же установлен рабочий каталог D:\thisFolder, то можно написать просто: Program.Exe Example.tsv

5 В данной папке содержится демонстрационный текстовый файл Example.tsv, с ним можно проверить работу приложения.

ПРИМЕЧАНИЯ

1 Для корректного отображения в консоли русского текста убедиться, что в консоли установлен шрифт Lucida Console. Для этого кликнуть правой клавишей мыши по шапке окна консоли, далее: Свойства - Шрифт - Lucida Console

2 Предполагается, что на рабочем компьютере файл компилятора csc.exe расположен по стандартному пути:
  C:\Windows\Microsoft.NET\Framework\v4.0.30319\csc
  Если это не так, открыть файл Start.bat, указать верный путь и сохранить файл. После запустить его.
